# 句子摘录

以前我不看文学，觉得没什么用，文绉绉的，现在我觉得这些精神性的东西虽然没办法直接和生产力挂钩的技术书籍相比，但是却可以在每个人内心需要慰藉时候，给予意想不到的慰藉。人的毕竟不是机器，人的内心，精神，心里所想， 也很重要。

这儿算是个人兴趣吧，自己把看到有意思的句子摘录到这儿来，可能来自于任何地方的句子，觉得有意思我就记录进来✨
